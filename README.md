# Cine-Taka-Toe
Nuestro proyecto consiste en un juego inspirado en el tres en raya pero en versión cine. Dos jugadores se enfrentan uno contra uno por hacer 3 en raya, cubriendo las casillas con películas que cumplan la condición de su fila y su columna. El juego utiliza la API de TMDB para verificar las películas y que estas cumplan las condiciones.
